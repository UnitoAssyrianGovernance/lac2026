# Cyclical Dynamics Between Climate and Population: a Multi-Proxy Approach from the South Levant

This repo contains the slides and code for the paper presented at the Landscape Archaeology Conference "Cyclical Dynamics Between Climate and Population: a Multi-Proxy Approach from the South Levant" on 20th of March 2026 in Bamberg. The repo is also archived on [Zenodo](https://doi.org/10.5281/zenodo.19126661) and the interactive presentation can be browsed at the following [link](https://www.andreatitolo.com/talks/2026-03-20-titolo-palmisano-lac/titolo_palmisano_lac2026#/title-slide).
